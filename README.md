# K.EStore

K.EStore is a simple and modern e-commerce front-end website built with HTML, CSS, and JavaScript.  
It includes a product grid, category sidebar, hero section, and a clean dark theme.

## Features
- Modern dark UI
- Responsive layout
- Product cards with hover effects
- Category sidebar
- Simple JavaScript product rendering
